# DeepLearn

这是一个“边学习深度学习、边实现项目、边编写教材”的长期实验型项目。

项目希望把整个学习过程沉淀成一套可复用的内容资产，包括：

1. 可运行的实战代码
2. 配套的中文教材
3. 学习过程中的经验、原则和方法论
4. 最终可辅助用户自主学习的 skill

## 实验意义

这个项目的实验意义不只是“训练一个模型”，而是在验证以下事情：

1. 是否可以把“边做项目边写教材”变成一种稳定学习方法
2. 是否可以把抽象算法知识沉淀成对初学者友好的学习路径
3. 是否可以通过代码、文档、复盘三线联动，加快理解速度
4. 是否可以把个人学习经验进一步抽象为可复用的 skill

## 最终目标

本项目最终希望形成三类产物：

1. 一套完整的深度学习中文教材
2. 一套可复用的教学型代码仓结构
3. 一套辅助用户自主学习的 skill 与 agent 协作原则

## 项目结构

### 教材目录

- `book/`
用于维护整套教材正文和教材级说明。

### 代码目录

- `code/`
用于承载与教材对应的代码实现、实验脚本、输出结果和公共模块。

### 方法论与元信息目录

- `meta/`
用于沉淀经验、原则、方法、沟通约定和未来 skill 化方向。
