# 图片引用与资源组织

本文件用于约定教材中的图片资源如何存放和引用。

## 1. 图片放在哪里

教材需要长期引用的图片，统一放在 `book/images/`。

建议按主题建子目录，例如：

1. `book/images/mnist/`
2. `book/images/cnn/`
3. `book/images/transformer/`

## 2. 什么时候要复制一份稳定副本

如果一张图片满足下面任一条件，就不要直接引用 `data/`、`outputs/` 里的原始文件，而应复制到 `book/images/`：

1. 这张图会在教材正文里长期出现
2. 这张图来自实验目录，后面可能会被清理或覆盖
3. 这张图会被多章复用

## 3. 正文里怎么引用

Markdown 正文优先使用相对路径引用 `book/images/` 下的文件。

例如，位于 `book/02-MNIST与神经网络入门/` 下的章节，要引用 `book/images/mnist/sample.png`，可以写成：

```md
![MNIST样本图](../images/mnist/sample.png)
```

## 4. 为什么不直接引用 data

`data/` 更接近下载缓存或实验产物。

如果正文直接引用 `data/` 下的图片，会有几个问题：

1. 数据目录后面可能被清理
2. 同一张图可能会被重新生成和覆盖
3. 教材资源和实验资源耦合过紧
