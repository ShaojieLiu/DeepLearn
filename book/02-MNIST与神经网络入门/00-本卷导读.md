# 本卷导读

这一卷是整套教材真正开始动手的地方。

如果说前言卷负责说明“为什么这样学”，基础准备卷负责说明“开始之前要准备什么”，那么这一卷负责回答：

“第一个真正要做的项目是什么，以及它应该怎么做。”

答案就是：`MNIST`。

但这里需要强调一点：`MNIST` 的价值不在于它有多先进，而在于它足够小、足够清晰、足够适合初学阶段建立第一个完整闭环。

在这一卷中，你会逐步完成以下事情：

1. 理解为什么从 `MNIST` 开始
2. 理解 `MNIST` 数据集和分类任务是什么
3. 跑通第一个最小可运行版本
4. 再进入从零实现和 `PyTorch` 对照

当前最先需要阅读的，是这一卷的前两篇：

1. [为什么从MNIST开始](./01-为什么从MNIST开始.md)
2. [MNIST数据集与分类任务](./02-MNIST数据集与分类任务.md)

这两篇的作用不是替你写代码，而是帮助你在写代码之前先知道：

1. 你现在在做什么任务
2. 这个任务为什么适合作为第一步
3. 你接下来应该先实现什么，不应该先实现什么

当前卷后面的“从零实现”和“`PyTorch` 对照”章节，会在你实际完成这一轮最小版本和下一阶段代码后继续展开。
